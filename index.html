HTML

HTML (HyperText Markup Language) es un lenguaje de programación que define la estructura y el significado de las páginas web. Es uno de los componentes más básicos de la web y es utilizado para estructurar y desplegar los contenidos de una página web, como párrafos, imágenes, tablas de datos y listas con viñetas.

CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño web que permite definir y crear la presentación visual de un documento escrito en un lenguaje de marcado como HTML. CSS se compone de archivos de programación que seleccionan elementos de una página y controlan su presentación.
